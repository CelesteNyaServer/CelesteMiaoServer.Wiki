- ## 技巧
- &emsp;&emsp;技巧（Techniques），是玛德琳用于提升她爬山能力的特殊移动方式。它们具有多种 用途和难度。很多技巧可以用来逃课，或是“利用”它们跳过几个部分而不必收集全部物品或规划整个流程。
