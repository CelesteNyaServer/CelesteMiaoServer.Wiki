## 群服开发目的
最初版本为GhostMod 目的只是为了能让大家玩蔚蓝开心一点 简简单单的服务器  
由于GhostMod过于老旧 并且新更新了**Celeste.Net** 所以决定在CelesteNet的基础上魔改开发出了现在的群服
